Build Prompts Programmatically and Write Responses to File. Steamlines programmatic AI iteration of files.

FilePromptForge is a command-line tool written in python that interacts with GPT AI and can read and write files.

It is made to simply the process of integrating chatbot interaction into your scripts or commands.


Command Line: build a prompt from files and save the response to a new file
Scripting: call it from your script to for a set of files, for each, build a prompt from files and save the response to rhe origanl file after commenting out the origanl text
